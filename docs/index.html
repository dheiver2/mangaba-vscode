Mangaba AI

O copiloto de IA brasileiro e soberano para VS Code

Chat, agente autônomo, edição de código, anexos (PDF/imagem) e RAG @codebase — funciona de fábrica ou 100% na sua própria infraestrutura (compatível com OpenAI).

Instalar via Open VSX Baixar .vsix Código-fonte
💬

Chat + streaming

Painel de chat integrado com seleção de modelo, visão e histórico de conversas.

🤖

Agente autônomo

Planejamento, terminal, auto-verificação e checkpoints com rollback.

🔎

RAG @codebase

Embeddings locais (ou via Ollama) para responder com contexto do seu projeto.

🔌

MCP

Conecte servidores MCP externos como ferramentas do agente.

📎

Anexos

Arraste PDF, imagem ou código — extração e análise automáticas.

🔒

Soberano por padrão

Funciona pronto de fábrica ou 100% na sua infra, sem depender de terceiros.

Como instalar

VSCodium, Cursor, Windsurf, Gitpod: procure por "Mangaba AI" na aba de extensões (registro Open VSX).

VS Code oficial: enquanto a publicação no Marketplace da Microsoft está em andamento, baixe o .vsix da release e rode:

code --install-extension mangaba-ai-0.11.12.vsix
